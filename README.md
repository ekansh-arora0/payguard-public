# PayGuard - AI Phishing Detection

Real-time protection against phishing, crypto scams, fake stores, and credential harvesting.

## Install

```bash
curl -fsSL https://raw.githubusercontent.com/ekansh-arora0/payguard-public/main/install.sh | bash
```

## Features

- Domain-tier URL analysis (lookalikes, brand impersonation, suspicious TLDs)
- Page structural analysis (forms, iframes, identity mismatch, payment analysis)
- JS obfuscation detection (trained on 1,978 real phishing kits)
- Behavioral text analysis (scam patterns, urgency, crypto scams)
