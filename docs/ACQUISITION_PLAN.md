# PayGuard Acquisition Plan
## Deterministic channels only — no social media dependency

---

## Strategy: Product-Led + Direct Outbound

Don't depend on viral content. Use **channels where you control the volume**.

---

## Channel 1: Chrome Extension (PASSIVE — highest ROI)

**What:** Browser extension that runs PayGuard detection automatically while browsing.

**Why it works:**
- Chrome Web Store has 3B+ users searching for security tools
- Free installs → users experience protection → convert to paid desktop app
- No marketing spend — the store drives traffic for you
- Reviews compound over time

**How:**
- Build Chrome extension using `page_analyzer.py` logic
- Analyze the current page URL on every navigation
- Show a green/red shield icon in the browser toolbar
- Free tier: URL-only analysis
- Paid tier: full HTML + JS + page analysis

**Timeline:** 1 week to build, 2 days to publish

**Expected:** 500-2000 installs/month with 0 marketing spend
